# Student Assignment Portal

This is a **Student Assignment Portal** built using Node.js, Express, and MongoDB.  
It allows admins to add students and faculty, faculty to assign tasks, and students to submit assignments.

## 游댢 Features

### Admin
- Add new students
- Add new faculty
- View all students & faculty
- Add quizzes

### Faculty
- Create assignments
- View submitted assignments
- View analytics
- Send reminders

### Students
- Register & login
- View assignments
- Submit assignments through files
- Submit quizzes

## 游 Tech Stack

- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Frontend:** HTML, CSS, JavaScript
